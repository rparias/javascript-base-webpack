# Javascript App

This application uses [Webpack](https://webpack.js.org/), [Babel](https://babeljs.io/).

To install the dependencies:

```
npm install
```

To start the app:

```
npm start
```

To run the tests:

```
npm test
```

To run the linter:

```
npm run lint
```

To format the code:

```
npm run format
```
